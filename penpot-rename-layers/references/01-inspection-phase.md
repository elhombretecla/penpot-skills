# Phase 0: Inspection — Layer Structure

Read-only phase. Collect all information needed to infer semantic names.

---

## What to Collect Per Shape

For each shape you need:

| Property | Why needed |
|----------|-----------|
| `id` | To build the rename map |
| `name` | Current name — detect auto-generated names vs. already meaningful ones |
| `type` | `frame`, `text`, `rect`, `ellipse`, `path`, `bool`, `image`, `group` |
| `x`, `y` | Position on canvas — detect header/footer by Y position |
| `width`, `height` | Size — detect buttons, inputs, small icons |
| `parentId` | Parent-child relationships for context |
| `children` | Child count and types — determines container role |
| `fills` | Background color — surface vs. transparent |
| `strokes` | Border-only shapes (likely inputs) |
| `fontFamily`, `fontSize`, `fontWeight` | Text role inference |
| `characters` | First N chars — identify what the text says |
| `borderRadius` | Pill shapes (≥999) vs. rounded cards |
| `componentId` | Is it a component instance? |
| `layout` | Has flex layout? What direction? |

## Auto-Generated Name Detection

Before renaming, classify each layer name as:

- **Auto-generated**: `Frame N`, `Rectangle N`, `Group N`, `Ellipse N`, `Text N`, `Path N` (where N is a number). These SHOULD be renamed.
- **Meaningful but non-semantic**: `Header`, `hero-section`, `CardComponent` — these COULD be renamed but flag for user decision.
- **Already semantic**: `nav`, `header`, `button-cta`, `h1-headline` — these should be PRESERVED (do not overwrite).

```javascript
function isAutoGenerated(name) {
  return /^(Frame|Rectangle|Group|Ellipse|Text|Path|Bool|Image|Component)\s+\d+$/i.test(name);
}

function isAlreadySemantic(name) {
  return /^(header|footer|nav|main|section|article|aside|form|button|input|select|label|h[1-6]|p|ul|ol|li|img|svg|dialog|figure|figcaption|a|span|small|caption|icon|div)[-_]?/i.test(name);
}
```

## Page Dimensions

To detect full-width frames, you need the page/viewport width. Estimate from the widest top-level frame:

```typescript
const page = penpot.currentPage;
const topLevel = page.findShapes().filter(s => !s.parentId);
const maxWidth = Math.max(...topLevel.map(s => s.width || 0));
return { estimatedPageWidth: maxWidth };
```

A frame is "full-width" if its width is ≥ 80% of the estimated page width.

## Component Instance Detection

Do not rename a shape's `name` if it is a **component instance** — renaming it only changes the instance override, not the master component. Note these separately in the rename plan as "instance — rename master to affect all."

```typescript
// shape.componentId exists for instances
// shape.isMainComponent is true for master component shapes
const isInstance = !!shape.componentId && !shape.isMainComponent;
```
